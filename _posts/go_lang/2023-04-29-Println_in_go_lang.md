## Println

It's a method provided by the package/library "fmt" that is used to display things on the console.

### Printing Simple Single Line Strings.

```
package main

import ("fmt")

func main(){
	fmt.Println("Hello Vipin.")
	fmt.Println("This is another line.")
}
```
